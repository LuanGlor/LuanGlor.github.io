关于我

关于我

你好！我是 LuanGlor，一名热爱编程和技术的开发者。目前专注于前端开发和全栈技术的学习与实践。

个人简介

  • 🎓 自学编程爱好者，从零开始学习前端技术
  • 💻 主要使用 JavaScript 生态，包括 Vue.js、React、Node.js
  • 📝 喜欢通过博客记录学习过程和分享技术心得
  • 🌟 相信编程不仅是一门技术，更是一种思维方式

关于这个博客

这个博客是我技术成长的见证，主要分享：

  • 前端开发：HTML、CSS、JavaScript 基础到进阶
  • 框架学习：Vue.js、React 等主流框架的使用经验
  • 后端技术：Node.js 服务器开发和 API 设计
  • 工具分享：开发工具、效率提升技巧
  • 学习笔记：编程题解、算法思路、技术踩坑记录

更新频率：每周至少一篇技术文章，分享最新的学习成果和项目经验。

技能栈

前端技术

  • 基础三剑客：HTML5、CSS3、JavaScript (ES6+)
  • 框架库：Vue.js 3、React 18、jQuery
  • 构建工具：Vite、Webpack、Parcel
  • 样式预处理：Sass、Less、Stylus

后端技术

  • Node.js：Express、Koa、NestJS
  • 数据库：MongoDB、MySQL、Redis
  • API 设计：RESTful、GraphQL

开发工具

  • 编辑器：VS Code、Typora
  • 版本控制：Git、GitHub
  • 包管理：npm、yarn、pnpm
  • 部署：Vercel、Netlify、Docker

学习计划

2026年计划深入学习：

  • 🚀 微前端架构和模块化
  • 📱 移动端开发 (React Native)
  • ☁️ 云服务和无服务器架构
  • 🤖 AI 辅助编程和自动化工具
